Let's Learn Pronouns!

When someone you know changes their preferred pronouns, it can sometimes be confusing and difficult to get used to.

Mistakes are understandable, but it's really important to practice and get it right to properly respect people and show you care about them.

If pronouns are tripping you up, try using this tool to correct yourself.

Which pronouns do you need some help with?

She/Her/Hers They/Them/Theirs He/Him/His

If you're not sure where to start, or you're not sure what someone's preferred pronouns are, here is a guide on how to use more gender neutral language in your daily speech:

Gendered Language Gender Neutral Language
he/she they
man/woman, female/male person
girl/boy child, young person, kid
ladies and gentlemen everyone, distinguished guests
mom/dad parent
sister/brother sibling
Mr., Mrs., Ms. Mx.
girlfriend/boyfriend, wife/husband partner, spouse
